Web Imposition

Booklet creation tool

Print booklets from home! Originally intended for zines, Web Imposition is a tool for taking PDFs and scaling their pages down onto sheets which can be folded and staple-bound.

  • Runs entirely in-browser — no installation or uploading to a server.
  • Cuts landscape images in half and places them onto two-page spreads.
  • Can process PDFs, individual images, and zip files.
